 для интеграции глификонов
 Скопируйте файлы шрифтов с: https://github.com/twbs/bootstrap-sass/tree/master/assets/fonts/bootstrap
 Скопируйте файл https://github.com/twbs/bootstrap-sass/blob/master/assets/stylesheets/bootstrap/_glyphicons.scss в свою папку bootstrap/scss
 Откройте файл scss/bootstrap/bootstrap.scss и запишите следующий код SCSS в конце этого файла:
 $bootstrap-sass-asset-helper: false;
 $icon-font-name: 'glyphicons-halflings-regular';
 $icon-font-svg-id: 'glyphicons_halflingsregular';
 $icon-font-path: '../fonts/';
 @import "glyphicons";
 Запустите: npm run dist чтобы перекомпилировать ваш код с Glyphicons.
